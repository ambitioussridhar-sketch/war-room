# The War Room

Stakeholder Strategy Simulator — v3.1

## What's New in v3.1
- **Streaming output**: Local AI now shows tokens as they arrive
- **Module preloading**: WebLLM module loads in background on page open
- **Cache detection**: Skips download if model already cached from previous session
- **Cancel button**: Stop the download mid-way if needed
- **Better progress**: Shows MB downloaded during model fetch

## Features
- **Rules Engine**: Instant, offline, deterministic simulation
- **Local AI**: Downloads Llama 3.2 1B to your browser, runs entirely offline
- **Claude API**: Optional cloud AI with your own API key
- **Stakeholder Dossier Builder**: Build detailed profiles with traits, stance, payoff structure
- **Simulation History**: Save and restore past simulations
- **Board Backup/Restore**: Export and import your stakeholder board
- **Fully Private**: No data leaves your browser (except Claude API calls)
